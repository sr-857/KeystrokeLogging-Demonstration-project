# AstraGuard Ethical Dashboard - Frontend

This is the React-based frontend for the Keystroke Logging Demonstration project.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS (Cyber-Noir Theme)
- **Icons**: Lucide-React
- **Charts**: Recharts
- **Date Handling**: date-fns

## Components
- `ConsentBanner`: Manages ethical consent state.
- `StatusCard`: Displays real-time logging status and counters.
- `LiveKeyStream`: Visualizes incoming keystroke events.
- `LogTable`: Searchable repository of captured logs.
- `AnalyticsPanel`: Advanced data visualization.
- `ScreenshotUploader`: Local proof-of-work preview.
- `AwarenessNotes`: Educational security content.

## Development
```bash
npm install
npm run dev
```
